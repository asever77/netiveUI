Introduction

Netive UI Guide는 그 동안 프로젝트를 진행하면서 필요했던 부분과 아쉬웠던 부분들을 지속적으로 추가하고 수정하여 만든 가이드이다.

HTML Template

기본적으로 HTML5로 마크업을 구성한다. 'id'사용은 최대한 지양하며 'class'로만 스타일을 준다.
기본구성의 class명은 접두사로 'base-'를 사용하며 사용 시 'id'처럼 유니크하게 사용한다.

<doctype html>
<html lang="ko">
	<head>
		<meta charset="utf-8">
		<meta http-equiv="X-UA-Compatible" content="IE=edge">
		<meta name="viewport" content="width=device-width, initial-scale=1.0, shrink-to-fit=no, user-scalable=yes">

		<title>page title</title>

		<link rel="stylesheet" href="/projectname/resources/css/ui.base.css">

		<script src="/projectname/resources/js/lib/jquery-1.12.4.min.js"></script>
		<script src="/projectname/resources/js/ui.global.js"></script>
		<script src="/projectname/resources/js/ui.common.js"></script>
	</head>
	<body>
		<div class="base-skip"></div>
		<div class="base-wrap">
			<header class="base-header">
				// 
			</header>
			<div class="base-body">
				<aside class="base-aside">
					// 사이드 영역 
				</aside>
				<main class="base-main">
					// 콘텐츠 영역
				</main>
			</div>
			<footer class="base-footer"></footer>
		</div>
		
		<script>
		;(function($, win, doc, undefined) {
			$(doc).ready(function(){
				// 스크립트영역  
			});
		})(jQuery, window, document);
		</script>
	</body>
</html>

Z-index

z-index를 3등급으로 나눈다.

  • A : 최상위 단위의 영역으로 구분
  • B : A등급영역 안에 들어가는 가장 큰 구분
  • C : A,B등급영역 안에 들어가는 가장 작은 단위
옵션 설명
Level class명 Default Explanation
A .base-skip 2 skip navigation
.base-wrap 0 base content
.ui-modal 300 ~ modal content
B .ui-drop-pnl 200 dropdown content
C .ui-tooltip 100 tooltip content
