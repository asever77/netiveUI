간격의 기본정의

기본 간격 단위

기본적인 간격은 8배수로 여백을 사용한다.

더 작은 단위가 필요한 경우 4배수를 사용한다.

Element 와 Element의 간격 우선순위

모든 간격의 기본은 상단(margin-top)기준으로 간격을 적용하며 Element 다음으로 오는 Element의 종류에 따라 간격을 달리한다.
