Buttons

Base Button

모든 버튼의 기본 형태이며 a, button 태그로만 구성합니다.

a tag 

// text button
<button type="button" 
	class="btn-base" 
	data-size="{ large | small }" 
	data-shape="{ round | no }" 
	data-color="{ primary | secondary | gray }
	data-icon="{ name }
	data-icon-ps="{ front | center | back }">
	<span>{name}</span>
</button>

// only icon button
<button type="button" 
	class="btn-base"  
	data-icon="{name}" 
	data-icon-ps="center">
	<span class="a11y-hidden">{name}</span>
</button>
<button type="button" 
	class="btn-base" 
	data-icon="{name}" 
	data-icon-ps="center"
	aria-label="{name}">
</button>
