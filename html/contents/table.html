caption

좀 더 자세하고 정확하게 표현하기 위해 strong 안에 테이블의 간략한 제목 기입과 '.ui-caption' 엘리먼트에 자동으로 테이블의 모든 th의 항목 텍스트를 기입하여 테이블을 설명한다.

번호, 제목, 이름, 아이디, 날짜 정보입니다. 

<caption>
	<strong>테이블의 제목</strong>
	<span class="ui-caption">[자동으로 테이블의 th의 제목이 들어감] 정보입니다.</span>
</caption>

col

col width 값은 css에서 정의하여 html 마크업과 css 스타일을 분리시킨다.

.tbl-base col:nth-child(1) { width:10rem; }
.tbl-base col.n2 { width:10rem; } 

<div class="tbl-base type-name">
	<table>
		<colgroup>
			<col class="n1">
			<col span="3" class="n2">
		</colgroup>
		<colgroup span="2" class="n3">
		...

.type-name col.n1 {
	width:5rem;
}
.type-name col.n2 {
	width:auto;
}
...

scope="colgroup" & <colgroup>

scope="colgroup" 열그룹은 colgroup 범위안에서 그룹핑을 한다. 사용 시 colgroup으로 영역 구분.


<colgroup>
	<col>
	<col>
	<col>
</colgroup>
<colgroup>
	<col>
	<col>
</colgroup>
<thead>
	<tr>
		<th scope="colgroup" colspan="3">A</th>
		<th scope="colgroup" colspan="2">B</th>
	</tr>
	<tr>
		<th scope="col">A-1</th>
		<th scope="col">A-2</th>
		<th scope="col">A-3</th>
		<th scope="col">B-1</th>
		<th scope="col">B-2</th>
	</tr>
</thead>

scope="rowgroup" & <tbody>

scope="rowgroup" 행그룹은 tbody 범위안에서 그룹핑을 한다. 사용 시 tbody으로 영역 구분.


<tbody>
	<tr>
		<th scope="rowgroup" rowspan="2">A</th>
		<th scope="row">A-a</th>
		<td>A-a-1</td>
	</tr>
	<tr>
		<th scope="row">A-b</th>
		<td>A-b-2</td>
	</tr>
</tbody>

<tbody>
	<tr>
		<th scope="rowgroup" rowspan="2">B</th>
		<th scope="row">B-a</th>
		<td>B-a-1</td>
	</tr>
	<tr>
		<th scope="row">B-b</th>
		<td>B-b-2</td>
	</tr>
</tbody>
