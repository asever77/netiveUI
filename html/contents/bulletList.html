Bullet List

'ul', 'ol', 'dl' 태그는 명확하게 목록의 성격과 제목을 가지고 있을 경우에만 사용한다.

그외의 경우에는 기본적으로 'block elements'을 활용한다.

Unordered list

순서가 없는 목록으로 점이나 사각형 등 도형이 앞에 붙는 목록.

block elements(div, p, ...)

콘텐츠영역에서 서술형 문단으로 블릿으로 구분지어야 할 경우 사용한다.
