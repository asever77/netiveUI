dropdown 기본 옵션
Options Type Default Description
id: [string] (필수) ui-drop의 아이디명 기입
ps: [string] 'BS' 4(상단, 하단, 왼쪽, 오른쪽) * 3(시작, 중간, 끝)의 조합으로 12가지 위치 설정
{T | B | R | L} + {S | C | E} or CC
hold: [boolean] true 포커스 홀드 여부
  • true : 'hold' 포커스 홀드
  • false : 'sense' 포커스 벗어날 때 자동으로 닫힘
area: [object] doc.querySelector('body') Ajax로 파일을 불러올 위치
src: [string] Ajax로 파일을 불러올 url경로
offset: [boolean] false 드롭다운 콘텐츠의 위치에 따라 설정
  • true : 위치값 절대값(offset)
  • false : 위치값 상대값(position)
callback: [function] false 오픈 시 콜백
closeback: [function] false 닫을 시 콜백

netive.dropdown.toggle


netive.dropdown.toggle({ 
	id:'uiDrop1',
	state: 'open' 
});
dropdown 기본 옵션
Options Type Default Description
id: [string] (필수) ui-drop의 아이디명 기입
state: [string]
  • 'open' : 열림
  • 'close' : 닫힘
