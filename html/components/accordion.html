$plugins.uiAccordion


$plugins.uiAccordion({ 
	id:'exeAcco1', 
	current:[0,2], 
	autoclose:true, 
	callback:function(v){console.log(v)} 
});
$plugins.uiAccordion 옵션 설명

옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-acco의 아이디명 기입
current: [null][array][string] null 처음 열린패널 선택
  • null : 모두 닫힘
  • 'all' : 모두 열림
  • [0,1] : 배열으로 열릴패널 선택
autoclose: [boolean] false 주변 패널 상태변화
  • true : 선택외의 다른 패널은 닫힘
  • false : 변화없음
callback: [function] function 콜백값으로 아이디와 현재선택 index값

$plugins.uiAccordionToggle


$plugins.uiAccordionToggle({ 
	id:'exeAcco1', 
	current:[0,2], 
	motion:false, 
	state:'open', 
	callback:function(v){console.log(v)} 
});
$plugins.uiAccordionToggle 옵션 설명

옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-acco의 아이디명 기입
current: [null][array][string] null 처음 열린패널 선택
  • null : 모두 닫힘
  • 'all' : 모두 열림
  • [0,1] : 배열으로 열릴패널 선택
autoclose: [boolean] false 주변 패널 상태변화
  • true : 선택외의 다른 패널은 닫힘
  • false : 변화없음
motion: [boolean] false 애니메이션 효과
  • true : 효과있음
  • false : 효과없음
state: [string] 'toggle' 상태 설정
  • 'toggle' : 토글방식
  • 'open' : 무조건 열림
  • 'close' : 무조건 닫힘
callback: [function] function 콜백값으로 아이디와 현재선택 index값

parameter 값으로 아코디언탭 선택방법


single 선택
/netiveUI/html/components/accordion.html?acco=1 

복수 선택
/netiveUI/html/components/tab.html?acco=exeAcco1*0+exeAcco2*1+exeAcco3*1 
탭아이디*선택+탭아이디*선택....
주의 : tab의 아이디를 '*'이 들어가는 이름으로 설정하면 안된다.
