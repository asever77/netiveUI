Accordion

Accordion UI는 한정된 영역에서 컨텐츠의 일부 영역만 노출시키고 다른 영역은 감춘다.
Tab보다는 의미적으로 더 작은 컨텐츠에 주로 사용되며 목록, 메뉴로도 사용 된다.

Javascript


netive.accordion.init({ 
	id:'exeAcco1', 
	current:[0,2], 
	callback:function(v){
		console.log(v);
	} 
});
netive.uiAccordion 옵션 설명
Options Type Default Explanation
id: [string] 'name' (필수) ui-acco의 아이디명 기입
current: [null][array][string] null 처음 열린패널 선택
  • null : 모두 닫힘
  • 'all' : 모두 열림
  • [0,1] : 배열으로 열릴패널 선택
autoclose: [boolean] false 주변 패널 상태변화
  • true : 선택외의 다른 패널은 닫힘
  • false : 변화없음
callback: [function] function 콜백값으로 아이디와 현재선택 index값

netive.uiAccordionToggle


netive.accordion.toggle({ 
	id:'exeAcco1', 
	current:[0,2], 
	callback:function(v){
		console.log(v);
	} 
});
netive.uiAccordionToggle 옵션 설명
Options Type Default Explanation
id: [string] 'name' (필수) ui-acco의 아이디명 기입
current: [null][array][string] null 처음 열린패널 선택
  • null : 모두 닫힘
  • 'all' : 모두 열림
  • [0,1] : 배열으로 열릴패널 선택
autoclose: [boolean] false 주변 패널 상태변화
  • true : 선택외의 다른 패널은 닫힘
  • false : 변화없음
motion: [boolean] false 애니메이션 효과
  • true : 효과있음
  • false : 효과없음
state: [string] 'toggle' 상태 설정
  • 'toggle' : 토글방식
  • 'open' : 무조건 열림
  • 'close' : 무조건 닫힘
callback: [function] function 콜백값으로 아이디와 현재선택 index값

parameter 값으로 아코디언탭 선택방법


단일 선택
/netiveUI/html/components/accordion.html?acco=1 

복수 선택
/netiveUI/html/components/accordion.html?acco=exeAcco1*1+exeAcco2*0+exeAcco3*2 
탭아이디*선택+탭아이디*선택....
주의 : accordion의 아이디를 '*'이 들어가는 이름으로 설정하면 안된다.
