Tooltip

툴팁(말풍선)은 마우스오버나 포커스가 가는 즉시 동작하여 보충설명을 보여준다.
항상 상위에 보이기 위해(z-index, overflow:hidden) baseLayer 위치에 생성한다.
기본적으로 자동으로 툴팁 실행위치를 계산하여 더 많은 공간이 있는 위치를 선택하여 보여준다.

Examples

Mouse Event
mouseover(show) = mouseout(hide)
click(show) = click(hide)
