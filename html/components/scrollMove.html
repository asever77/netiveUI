Source Code

HTML


<div class="ui-scrollmove" data-area="name">
	<div class="ui-scrollmove-item" data-name="name1">...</div>
	...
</div>

<button type="button" class="ui-scrollmove-btn" data-area="name" data-name="name1" data-align="center" data-callback="testscroll">...</button>

//callback
netive.callback.testscroll = function(){
	console.log('callback');
};

Javascript


netive.scroll.init();

netive.scroll.move({ 
	selector: document.querySelector('.ui-scrollmove[data-area="test1"]'),
	focus: document.querySelector('.ui-scrollmove-item[data-area="3"]'),
	top: 0,
	left: -300,
	add: 0,
	align: 'default',
	callback: function(){ console.log('test'); }
});
옵션 설명
Options Type Default Description
selector: [object] document.querySelector('html, body') 스크롤 영역 지정
top: [number] 0 세로 이동할 위치값
left: [number] 0 가로 이동할 위치값
add: [number] 0 이동 시 이동값 추가
align: [string] 'default' 스크롤 아이템 기준위치
  • 'default' : top,left
  • 'center' : center,middle
focus: [object] false 스크롤 완료 후 포커스 갈 위치의 id or $(...)
callback: [function/string]
  • 이동 시 : 스크롤 좌표값, complete = false
  • 완료 시 : 이동할 포커스 아이디명, complete = true
