Javascript

$plugins.scroll.move({ ... })


$plugins.scroll.move({ 
	selector: $('name'), 
	value: 0,
	speed: 0,
	callback: false,
	ps: 'top', //'top/center/left'
	addLeft: 0,
	focus: false,
	
});
옵션 설명
옵션명 유형 기본 설명
value: [number] 0 스크롤이 갈 위치값
speed: [number] 300 스크롤속도
addLeft: [number] 0 가로 이동 시 이동값 추가
target: [string][object] 'html, body' 스크롤 영역으로 해당 아이디명 기입. 기본으로 html,body로 설정됨
ps: [string] 'top' 스크롤 방향
  • 'top' : 상하 이동
  • 'left' : 좌우 이동
  • 'center' : 좌우 중앙 이동
focus: [object] false 스크롤 완료 후 포커스 갈 위치의 id or $(...)
callback: [function]
  • 이동 시 : 스크롤 좌표값, complete = false
  • 완료 시 : 이동할 포커스 아이디명, complete = true
