Javascript

$plugins.scroll.move({ ... })


$plugins.scroll.move({ 
	selector: $('name'), 
	value: 0,
	speed: 0,
	callback: false,
	ps: 'top', //'top/center/left'
	add: 0,
	focus: false,
	
});
옵션 설명
옵션명 유형 기본 설명
selector: [object] $('html, body') 스크롤 영역 지정
value: [number] 0 이동할 위치값
speed: [number] 300 이동속도
addLeft: [number] 0 가로 이동 시 이동값 추가
ps: [string] 'top' 스크롤 방향
  • 'top' : 상하 이동
  • 'left' : 좌우 이동
  • 'center' : 좌우 중앙 이동
focus: [object] false 스크롤 완료 후 포커스 갈 위치의 id or $(...)
callback: [function]
  • 이동 시 : 스크롤 좌표값, complete = false
  • 완료 시 : 이동할 포커스 아이디명, complete = true
