$plugins.uiScroll


$plugins.uiScroll({ value:0, speed:300, ps:'top', target:'html,body' focus:'baseHeader', callback:function(v){console.log(v)} });
옵션 설명
옵션명 유형 기본 설명
value: [number] 0 스크롤이 갈 위치값
target: [string] 'html, body' 스크롤 영역으로 해당 아이디명 기입. 기본으로 html,body로 설정됨
ps: [string] 'top' 스크롤 방향
  • 'top' : 상하 이동
  • 'left' : 좌우 이동
speed: [number] 300 스크롤속도
focus: [string] false 스크롤 완료 후 포커스 갈 위치의 아이디명
callback: [function]
  • 이동 시 : 스크롤 좌표값, complete = false
  • 완료 시 : 이동할 포커스 아이디명, complete = true
