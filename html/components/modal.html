Javascript

$plugins.uiModal


$plugins.uiModalOpen({ 
	id:'IDname', 
	type: 'normal',
	wrap: $('body'),
	moblieFull: false,
	ps: 'center',
	src: false,
	remove: false,
	modalWidth: false,
	modalHeight: false,
	innerScroll: false,
	mg: 10,
	callback:function() { console.log('open callback'); },
	closeCallback:function() { console.log('open callback'); },
	endfocus:false 
});
$plugins.uiModal 옵션 설명

옵션명 유형 기본 설명
id: [string] (필수) ui-modal의 아이디명 기입
type: [string] 'normal'
  • 'normal' : 콘텐츠 모달
  • 'alert' : 시스템 팝업
  • 'confirm' : 시스템 팝업
ps: [string] 'center' 모달 레이어 팝업 생성될 위치 설정
  • 'top' : 상단 위치
  • 'center' : 중앙 위치
  • 'bottom' : 하단 위치
mobileFull: [boolean] false mobile에서 전체 사이즈 모달 레이어 팝업 설정
  • true : 전체화면
  • false : 기본 사이즈
src: [string] Ajax로 파일을 불러올 url경로
endfocus: [string] document.activeElement 모달 레이어 팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달 레이어 팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정
modalWidth: [number] 모달 레이어 팝업 가로 크기
옵션이 없을 경우 모달 팝업의 크기
modalHeight: [number] 모달 레이어 팝업 세로 크기
옵션이 없을 경우 모달 팝업의 크기
callback: [function] false 모달 레이어 팝업 오픈 시 콜백값 실행
closeCallback: [function] false 모달 레이어 팝업 닫을 시 콜백값 실행

$plugins.modal.system


// 확인 
$plugins.modal.system({ 
	type : 'alert',
	btn_alert : '확인',
	cont_text : '확인하세요.',
	state : '알림',
	confirmCallback: function(){console.log('alert 확인')} 
});

// 선택 
$plugins.modal.system({ 
	type : 'alert',
	btn_alert : '확인',
	cont_text : '오류가 있습니다.',
	state : '오류',
	confirmCallback: function(){console.log('alert 확인')} 
});

// 오류 
$plugins.modal.system({ 
	type : 'confirm',
	btn_confirm_yes : '확인',
	btn_confirm_no : '취소',
	cont_text : '둘중에 하나를 선택하세요. <br> 오직 하나만~',
	state : '선택',
	confirmCallback: function(){console.log('confirm 확인')},
	cancelCallback: function(){console.log('confirm 취소')} 
});

$plugins.uiModalOpen

$plugins.uiModal (open:false) 옵션을 준 상태 모달 레이어 팝업 오픈 


$plugins.uiModalOpen({ id:'modal_name' });

$plugins.uiModalResize

모달 레이어 팝업안에 컨텐츠의 동적 생성으로 사이즈 변경이 필요할 시 실행


$plugins.uiModalResize({ id:'modal_name' });

$plugins.uiModalClose

모달 레이어 팝업 닫을 시 사용


$plugins.uiModalClose({ 
	id:'modal_name', 
	endfocus:'idname', 
	callback:function(v){console.log(v)} 
});
$plugins.uiModalClose 옵션 설명

옵션명 유형 기본 설명
id: [string] 'modal_name' (필수) ui-modal의 아이디명 기입
remove: [boolean] false 모달팝업 콘텐츠 삭제 여부
  • true: 코드삭제
  • false: 미삭제
endfocus: [string] document.activeElement 모달팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정
callback: [function] 모달팝업 닫을 시 콜백함수 실행
