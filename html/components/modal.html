Javascript

base show


//base modal
netiv.modal.show({ 
	id:'IDname', 
	type: 'normal',
	full: false,
	ps: 'center',
	src: false,
	remove: false,
	width: false,
	height: false,
	mg: 10,
	callback: function() { console.log('callback'); },
	endfocus: false //document.querySelector('...'),
});
netive.uiModal 옵션 설명
Options Type Default Description
id: [string] (필수) ui-modal의 아이디명 기입
type: [string] 'normal'
  • 'normal' : 콘텐츠 모달
  • 'alert' : 시스템 팝업
  • 'confirm' : 시스템 팝업
ps: [string] 'center' 모달 레이어 팝업 나타날 위치 설정
  • 'top' : 상단 위치
  • 'center' : 중앙 위치
  • 'bottom' : 하단 위치
full: [boolean] false mobile에서 전체 사이즈 모달 레이어 팝업 설정
  • true : 전체화면
  • false : 기본 사이즈
src: [string] Ajax로 파일을 불러올 url경로
remove: [boolean] false 코드 삭제 여부
width: [number] 모달 레이어 팝업 가로 크기
옵션이 없을 경우 모달 팝업의 크기
height: [number] 모달 레이어 팝업 세로 크기
옵션이 없을 경우 모달 팝업의 크기
mg: [number] 10 모달창 상하 여백 크기
callback: [function] false 모달 레이어 팝업 오픈 시 콜백값 실행
endfocus: [string] document.activeElement 모달 레이어 팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달 레이어 팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정

base hide

모달 레이어 팝업 닫을 시 사용


netive.modal.hide({ 
	id: 'modal_name', 
	remove: false,
	endfocus: document.querySelector('...'),
	callback: function(v){ console.log(v) }
});
netive.modal.hide 옵션 설명
Options Type Default Description
id: [string] (필수) ui-modal의 아이디명 기입
remove: [boolean] false 모달팝업 콘텐츠 삭제 여부
  • true: 코드삭제
  • false: 미삭제
endfocus: [string] document.activeElement 모달팝업 닫을 시 포커스가 갈 위치의 아이디 값 설정
기본으로 모달팝업 띄우기 전 포커스로 설정되나 그 값도 없을 경우 body로 설정
callback: [function] 모달팝업 닫을 시 콜백함수 실행

system show


netive.modal.show({ 
	type: 'alert', //'confirm'
	ps: 'center',

	sMessage: '',
	sBtnConfirmTxt: 'Ok',
	sBtnCancelTxt: 'Cancel',
	sZindex: false,
	sClass: 'type-system',
	sConfirmCallback: function() { console.log('callback'); },
	sCancelCallback: function() { console.log('callback'); }
});
netive.uiModal - system modal option 설명
Options Type Default Description
sMessage: [string] 메시지 입력. 태그가능
sBtnConfirmTxt: [string] confirm 버튼 명
sBtnCancelTxt: [string] cancel 버튼 명
sZindex: [string] zindex 값 지정
sClass: [string] class 추가
sConfirmCallback: [string] confirm 클릭 시 콜백
sCancelCallback: [string] cancel 클릭 시 콜백

system hide

모달 레이어 팝업 닫을 시 사용


netive.modal.hideSystem();
