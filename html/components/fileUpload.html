file upload

Button Type

button으로 스타일을 적용할수 있게 자동으로 input type="file"을 생성


<div class="ui-file" id="addfile1">
    <button type="button" class="btn-base ui-file-btn"><span>첨부파일</span></button>
</div>

$plugins.uiFileUpload


// 싱글 파일업로드
$plugins.uiFileUpload({ id:'name', accept:'image/*' });

// 멀티 파일업로드
$plugins.uiFileUpload({ id:'name', multi:true });
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-file의 아이디명 기입
accept: [string] '' 파일업로드 유형선택 속성 참고
multi: [boolean] false 멀티 업로드 선택
  • true : 멀티업로드
  • false : 단일 업로드
