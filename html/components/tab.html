Source Code

HTML


<div class="ui-tab" id="exeTab1">
	<div class="ui-tab-btns">
		<div class="btn-wrap">
			<button type="button" data-tab="0" class="ui-tab-btn"><span>탭1</span></button>
			...
		</div>
	</div>
	<div class="ui-tab-pnls">
		<section class="ui-tab-pnl" data-tab="0">
			<h3 class="ui-tab-tit">탭2 title</h3>
		</section>
		...
	</div>
</div>

Javascript


netive.tab.init({
	id:'exeTab1', 
	current:0, 
	onePanel: false,
	effect: false,
	align : 'center',
	callback:function(v){ 
		console.log(v); 
	}
});

netive.tab.toggle({ 
	id:'exeTab1', 
	current:0, 
	callback:function(v){ 
		console.log(v); 
	}
});
옵션 설명
Options Type Default Explanation
id: [string] 'name' (필수) ui-tab의 아이디명 기입
current: [number] 0 열린패널 선택
callback: [function] function 콜백값으로 아이디와 현재선택 index값

URl parameter


single 선택
?tab=1 
탭아이디*선택

dual 선택
?tab=exeTab1*0+exeTab1_2*1+exeTab2*1 
탭아이디*선택+탭아이디*선택....
주의 : tab의 아이디를 '*'이 들어가는 이름으로 설정하면 안된다.
