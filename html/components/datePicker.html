옵션 설명
태그 속성 값 Explanation
type="date" min YYYY-MM-DD 최소선택 년월일
max YYYY-MM-DD 최대선택 년월일
data-period 'start' or 'end' 기간선택일 경우 시작일과 종료일 설정
id (multi 일 경우) 유니크한 아이디이름 A + '_end' 시작일 아이디에 _end 를 붙인 이름 규칙으로 아이디 사용

Javascript


$plugins.datepicker.init({
	id: 'input id name',
	date: 'YYYY-MM-DD',
	min: 'YYYY-MM-DD',
	max: 'YYYY-MM-DD',
	title: 'title name',
	period: 'start' or 'end'
});

$plugins.datepicker.confirm({
	id: 'input id name',
	callback: function(){
		console.log('callback confirm');
	}
});

$plugins.datepicker.destoroy({
	id: 'input id name',
	callback: function(){
		console.log('callback destoroy');
	}
});
옵션 설명
Options Type Default Explanation
period : [string] start 'start', 'end'
title : [string] input title 달력의 제목 설정
callback: [function] false 날짜를 선택했을 때 콜백값으로 {id: "uiDatePicker1Inp", value: "2020-06-16", name: undefined}
