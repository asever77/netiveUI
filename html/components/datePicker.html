옵션 설명
태그 속성 값 Explanation
.ui-datepicker data-period false 기간설정. script에서 옵션값 대신 줄 수 있음
data-dual false 달력 2개월 보이게 설정. script에서 옵션값 대신 줄 수 있음
data-title false 달력 제목 설정. script에서 옵션값 대신 줄 수 있음
input type="tel" data-min 2013-01-01 최소선택 년월일
data-max 2020-01-01 최대선택 년월일
id 유니크한 아이디이름 A
id (multi 일 경우) 유니크한 아이디이름 A + '_end' 시작일 아이디에 _end 를 붙인 이름 규칙으로 아이디 사용

Javascript

$plugins.uiDatePicker


//single
$plugins.uiDatePicker({ 
	selector: '.ui-datepicker', // '#uiDatePicker1'
	dual: false,
	period: false,
	title: false, //'datepicker title'
	date_split: '-', //'.'
	shortDate: false, //false:YYYYMMDD or true:DDMMYYYY
	dateMonths: new Array('01', '02', '03', '04', '05', '06', '07', '08', '09', '10', '11', '12'),
	weekDay: new Array('일', '월', '화', '수', '목', '금', '토'),

	callback: function(v){ console.log(v); },
	openback: function(){ console.log('open callback'); },
	closeback: function(){ console.log('close callback'); }
});
옵션 설명
Options Type Default Explanation
selector: [string] 'name' .class나 #아이디 값
dual: [boolean] false true 일 경우 2개월이 보임
period : [boolean] false true 일 경우 기간 선택으로 변경
title : [string] input title 달력의 제목 설정
date_split : [string] '-' '-' : 2020-05-10
'.' : 2020.05.10
shortDate : [boolean] false false:YYYYMMDD or true:DDMMYYYY
dateMonths : [boolean] new Array('01', '02', '03', '04', '05', '06', '07', '08', '09', '10', '11', '12') 월 텍스트
weekDay : [boolean] new Array('일', '월', '화', '수', '목', '금', '토') 요일 텍스트
callback: [function] false 날짜를 선택했을 때 콜백값으로 {id: "uiDatePicker1Inp", value: "2020-06-16", name: undefined}
openback: [function] false 달력 오픈 시 콜백
closeback: [function] false 달력 닫을 시 콜백
