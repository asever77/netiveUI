특정 영역 안에서만 이동

$plugins.uiFloating

필수 옵션과 선택 옵션의 정보

옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) fix follow를 적용 시킬 영역(ui-floating)의 id 명
ps: [string] 'top' 고정될 위치 설정
  • 'top' : 상단
  • 'bottom' : 하단
fix: [boolean] true 고정되어 시작할지 여부
  • true : 고정에서 기본
  • false : 기본에서 고정
add: [string] 'name' 고정되어 있는 오브젝트가 이미 있다면 그위치 다음으로 위치하기 위해 해당 영역의 아이디명
callback: [function] function 콜백값으로 아이디와 현재스크롤위치값, 상태변경 위치값
