Components

Brick List

벽돌을 쌓아 올리듯 비어있는 공간이 없이 정렬시키는 리스트이다.
가로 길이를 동일한 길이로 기준으로 맞추어 정리되며 옵션에서 margin 값으로 간격조절이 가능하다.

Examples

css 기준

ui-bricklist-item의 정해진 css width 값 기준으로 정렬

option 기준

option의 col 값 기준으로 정렬

HTML Structure


<div class="ui-bricklist" role="list" id="uiBrickList1">
	<ul class="ui-bricklist-wrap">
		<li class="ui-bricklist-item" role="listitem">
			<div><img data-src="/netiveUI/resources/img/dummy/@iu1.jpg" alt=""></div>
			<strong>title</strong>
		</li>
		...
	</ul>
</div>

Javascript

$plugins.uiBrickList


$plugins.uiBrickList({ 
	id:'uiBrickList1', 
	fixCol: 3, 
	response:true  
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-bricklist의 아이디명 기입
fixCol: [number] n 세로단의 고정갯수
response: [boolean] true resize 시 재구성여부

$plugins.uiBrickListItem

동적으로 추가되는 리스트가 있을 경우 사용


$plugins.uiBrickListItem({ 
	id:'uiBrickList1' 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-bricklist의 아이디명 기입
