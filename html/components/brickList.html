Base Html Markup


<div class="ui-bricklist" role="list" id="uiBrickList1">
	<ul class="ui-bricklist-wrap">
		<li class="ui-bricklist-item" role="listitem">
			<div><img src="/netiveUI/resources/img/dummy/@iu1.jpg" alt=""></div>
			<strong>title</strong>
		</li>
		...
	</ul>
</div>

$plugins.uiBrickList


$plugins.uiBrickList({ 
	id:'uiBrickList1', 
	margin:10, 
	response:true  
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-bricklist의 아이디명 기입
margin: [number] 10 item간의 간격
response: [boolean] true resize 시 재구성여부

$plugins.uiBrickListAdd

동적으로 추가되는 리스트가 있을 경우 사용


$plugins.uiBrickListAdd({ 
	id:'uiBrickList1' 
});
옵션 설명
옵션명 유형 기본 설명
id: [string] 'name' (필수) ui-bricklist의 아이디명 기입
