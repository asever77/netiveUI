Select Design

CSS Design

CSS로 스타일 표현하나 IE등 일부 브라우저에서는 지원을 안함.

기본 스타일 

<select title="고스트 선택">
	<option value="1">먹깨비</option>
	<option value="2">캐스퍼</option>
</select>

Script Select

모든 브라우저에서 동일한 UI와 디자인으로 접근 가능. option도 디자인 표현이 가능

지역선택 

<label for="uiSel1" class="lb-tit">레이블</label>
<div class="ui-select">
	<select title="선택" id="uiSel1">
		<option value="1">고스트1</option>
		<option value="2">고스트2</option>
	</select>
</div>
