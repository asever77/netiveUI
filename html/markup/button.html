버튼

Button Base

기본적으로 <btn-wrap> </btn-wrap>로 버튼요소를 감싸줍니다. 버튼태그로는 <a>, <button> 태그를 사용 하며,
<a>는 링크요소버튼으로 'target="" ', 'title="" '속성을 함께 사용하고, <button>는 페이지내에서 이뤄지는 동작관련 버튼으로써 사용한다.

a 버튼 

<div class="btn-wrap">
	<button class="btn-base" type="button"><span>버튼</span></button>
	<a class="btn-base" href="#none"><span>버튼</span></a>
</div>

Button Size

버튼의 사이즈는 btn-s, btn-l, btn-xl를 추가하면 됩니다.

a 버튼 small size
a 버튼 기본(regular) size
a 버튼 midium size
a 버튼 large size 
	
<div class="btn-wrap">
	<button class="btn-base btn-s" type="button"><span>버튼s사이즈</span></button>
	<a class="btn-base btn-s" href="#none"><span>버튼s사이즈</span></a>
	
	<button class="btn-base btn-l" type="button"><span>버튼l사이즈</span></button>
	<a class="btn-base btn-l" href="#none"><span>버튼l사이즈</span></a>

	<button class="btn-base btn-xl" type="button"><span>버튼xl사이즈</span></button>
	<a class="btn-base btn-xl" href="#none"><span>버튼xl사이즈</span></a>
</div>

Button 좌우정렬

버튼의 좌우정렬 기본은 중앙정렬이며, 부모요소에 좌측정렬은 txt-l,클래스를, 우측정렬은 txt-r 클래스를 추가하면 됩니다.

a 버튼 좌측
a 버튼 센터
a 버튼 우측 
	
<div class="btn-wrap txt-l">
	<button class="btn-base" type="button"><span>버튼 좌측</span></button>
	<a class="btn-base" href="#none"><span>버튼 좌측</span></a>
</div>

<div class="btn-wrap">
	<button class="btn-base" type="button"><span>버튼</span></button>
	<a class="btn-base" href="#none"><span>버튼</span></a>
</div>

<div class="btn-wrap txt-r">
	<button class="btn-base" type="button"><span>버튼 우측</span></button>
	<a class="btn-base" href="#none"><span>버튼 우측</span></a>
</div>

Button 스타일

버튼의 메인 스타일은 버튼에 btn-main@,클래스를 메인 서브에는 btn-main@-sub 추가하면됩니다.
그외 스타일은 btn-color@ 클래스를 추가하면 됩니다.
(@는 숫자)

다음 (메인 스타일)
다음 (메인 스타일2)
a 버튼 스타일2 
	
<div class="btn-wrap">
	<button class="btn-main-sub" type="button"><span>버튼</span></button>
	<a class="btn-main" href="#none"><span>버튼</span></a>
</div>

<div class="btn-wrap">
	<button class="btn-main2" type="button"><span>버튼</span></button>
	<a class="btn-main2-sub" href="#none"><span>버튼</span></a>
</div>

<div class="btn-wrap">
	<button class="btn-mian btn-color1" type="button"><span>버튼</span></button>
</div>

<div class="btn-wrap">
	<a class="btn-mian btn-color2" href="#none"><span>버튼</span></a>
</div>
