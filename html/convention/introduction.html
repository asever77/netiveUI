Z-index

체계적인 z-index를 구성하기 위해 크게 A, B Level로 나눈다.

옵션 설명
Level Name Default Description
A .base-wrap 0 base content (Level B group)
.ui-modal 100 ~ 199 modal content
.ui-drop-pnl 200~299 dropdown content
.ui-tooltip 300~399 tooltip content
.base-skip 500 skip navigation
B .base-wrap > .base-header 20 header contents
.base-wrap > .base-body 10 main contents
.base-wrap > .base-footer 0 footer content
