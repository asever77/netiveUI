Toggle Event

기본적인 토글 이벤트 관련 모듈로 data-toggle-object="{name}", data-toggle-target="{name}"로 연결되어 data-toggle-state="{true | false}" 상태값 변경

    
    <button type="button" data-toggle-object="a" data-callback="toggle_a" data-toggle-state="false">
        <span>버튼</span>
    </button>  
    <div data-toggle-target="a" data-toggle-state="false"></div>

    <script type="module">
        import Toggle from '../assets/js/components/toggle.js';

        UI.exe.toggle = new Toggle();
        UI.callback.callbackname = (result) => {
            console.log('callback', result);
        }
    </script>
